# Identité visuelle

Le site vitrine doit reprendre l'univers naturel, agricole et professionnel de Dream Market, tout en donnant une impression plus institutionnelle pour présenter Dream Field.

## Ton visuel

Mots clés :

- Naturel.
- Local.
- Fiable.
- Moderne.
- Agricole.
- Accessible.
- Professionnel.

À éviter :

- Un style trop enfantin.
- Trop de couleurs vives.
- Trop d'animations.
- Un design trop chargé.
- Des textes trop longs sur la page d'accueil.

## Palette principale

Couleurs issues du thème de l'application :

- Vert principal : `#283106`
- Vert sombre : `#1B2504`
- Vert secondaire : `#777E5C`
- Vert clair : `#D1D8BD`
- Beige accent : `#C7C2AB`
- Fond doux : `#DFE0DC`
- Blanc carte : `#FFFFFF`
- Succès / feuille : `#4CAF50`
- Orange accent agricole : `#FF9800`

## Usage recommandé

Vert principal `#283106` :

- Header.
- Titres importants.
- Boutons principaux.
- Footer.

Vert secondaire `#777E5C` :

- Sous-titres.
- Texte secondaire.
- Icônes discrètes.

Vert clair `#D1D8BD` :

- Fonds de sections.
- Cartes douces.
- Badges.

Beige `#C7C2AB` :

- Séparateurs.
- Éléments décoratifs.
- Fonds de citations.

Blanc `#FFFFFF` :

- Cartes.
- Blocs de contenu.
- Zones de capture d'écran.

## Typographies

Recommandation web :

- Titres : **Poppins** ou **Sora**.
- Texte courant : **Inter** ou **Nunito Sans**.
- Alternative simple : utiliser uniquement **Inter** pour tout le site.

Hiérarchie :

- H1 desktop : 56 à 64 px.
- H1 mobile : 36 à 42 px.
- H2 desktop : 36 à 44 px.
- H2 mobile : 28 à 34 px.
- Texte courant : 16 à 18 px.
- Texte secondaire : 14 à 16 px.

## Style des composants

Boutons :

- Rayon : 999 px ou 14 px selon le style.
- Bouton principal : fond vert `#283106`, texte blanc.
- Bouton secondaire : bordure vert principal, fond transparent ou blanc.

Cartes :

- Rayon : 18 à 28 px.
- Ombre douce.
- Bordure légère `rgba(40, 49, 6, 0.08)`.
- Beaucoup d'espace blanc.

Images :

- Coins arrondis.
- Ratio homogène.
- Lumière naturelle si photos terrain.
- Captures mobile encadrées dans des mockups de téléphone.

## Style du hero

Composition recommandée :

- À gauche : titre, sous-titre, boutons, preuves rapides.
- À droite : mockup téléphone avec capture de l'application.
- Fond : dégradé doux vert clair vers blanc cassé.

Exemple de fond :

```css
background: linear-gradient(135deg, #DFE0DC 0%, #D1D8BD 45%, #FFFFFF 100%);
```

## Iconographie

Utiliser des icônes simples, linéaires :

- Feuille.
- Panier.
- Camion ou livraison.
- Téléphone.
- Bouclier.
- Utilisateurs.
- Carte ou localisation.
- Graphique pour l'espace admin.

## Direction photo

Photos idéales :

- Produits frais bien éclairés.
- Producteurs ou équipe sur le terrain.
- Clients utilisant l'application.
- Moments de préparation ou livraison.
- Photo de groupe de l'équipe Dream Field.

Style :

- Naturel et réel.
- Pas trop corporate générique.
- Montrer la RDC et le contexte local si possible.

## Ton éditorial

Le ton doit être :

- Clair.
- Rassurant.
- Professionnel.
- Humain.
- Orienté bénéfices.

Exemple :

Préférer :

> Commandez vos produits agricoles frais en quelques étapes.

Éviter :

> Notre solution révolutionnaire digitalise de manière disruptive la chaîne de valeur agroalimentaire.
