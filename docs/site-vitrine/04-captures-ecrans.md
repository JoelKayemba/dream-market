# Guide des captures d'écran

Ce fichier liste les captures à fournir pour construire un site vitrine convaincant.

## Format recommandé

Pour chaque capture :

- Format : PNG.
- Qualité : haute résolution.
- Appareil : idéalement iPhone récent et Android récent.
- Fond : sans notifications système visibles.
- Langue : français.
- Données : utiliser des produits réalistes, pas de données sensibles.
- Barre de statut : propre, batterie et réseau normaux.

## Captures prioritaires

### 1. Accueil de l'application

Utilisation sur le site :

- Hero principal.
- Section "Aperçu de l'application".

Ce qu'on doit voir :

- Logo ou titre Dream Market.
- Sections produits.
- Interface propre et attractive.

Nom conseillé :

`01-app-accueil.png`

### 2. Catalogue produits

Utilisation :

- Section fonctionnalités.
- Bloc "Produits locaux".

Ce qu'on doit voir :

- Plusieurs cartes produits.
- Prix.
- Images produits.
- Recherche ou filtres si possible.

Nom :

`02-catalogue-produits.png`

### 3. Détail produit

Utilisation :

- Montrer la confiance et les informations produit.

Ce qu'on doit voir :

- Grande image produit.
- Nom.
- Prix.
- Ferme ou fournisseur.
- Bouton d'ajout au panier.

Nom :

`03-detail-produit.png`

### 4. Panier

Utilisation :

- Section "Commande simple".

Ce qu'on doit voir :

- Produits ajoutés.
- Quantités.
- Totaux.
- Frais de livraison si disponible.

Nom :

`04-panier.png`

### 5. Checkout ou confirmation

Utilisation :

- Rassurer sur le processus de commande.

Ce qu'on doit voir :

- Adresse.
- Moyen de paiement hors app si applicable.
- Résumé clair.

Nom :

`05-checkout.png`

### 6. Historique ou détail commande

Utilisation :

- Section suivi.

Ce qu'on doit voir :

- Statut de commande.
- Informations de livraison.
- Total.

Nom :

`06-suivi-commande.png`

### 7. Services agricoles

Utilisation :

- Section "Services".

Ce qu'on doit voir :

- Liste des services.
- Cartes.
- Bouton de détail ou réservation.

Nom :

`07-services.png`

### 8. Profil utilisateur

Utilisation :

- Section compte et confiance.

Ce qu'on doit voir :

- Informations utilisateur non sensibles.
- Accès historique, support, paramètres.

Nom :

`08-profil.png`

### 9. Espace admin

Utilisation :

- Section Dream Field ou back-office.

Ce qu'on doit voir :

- Dashboard admin.
- Commandes.
- Produits ou statistiques.

Attention :

Masquer toute donnée sensible avant publication.

Nom :

`09-admin-dashboard.png`

## Captures entreprise

### Photos Dream Field

À fournir si possible :

- Équipe en groupe.
- Responsable ou fondateur.
- Bureau ou lieu de travail.
- Terrain agricole.
- Produits préparés.
- Livraison ou relation client.

Noms conseillés :

- `team-dream-field.jpg`
- `founder.jpg`
- `terrain-agricole.jpg`
- `produits-frais.jpg`
- `preparation-commandes.jpg`

## Placement sur le site

Hero :

- 1 capture principale de l'accueil dans un mockup téléphone.

Section fonctionnalités :

- 3 captures en cartes : catalogue, panier, suivi.

Section application :

- Slider ou grille de 6 captures.

Section Dream Field :

- Photo d'équipe ou photo terrain.

Section confiance :

- Photo réelle + chiffres clés + valeurs.

## Checklist avant envoi

- Aucune adresse client réelle visible.
- Aucun numéro client réel visible.
- Aucun email client visible.
- Aucun token, identifiant technique ou donnée admin sensible.
- Produits et prix cohérents.
- Captures nettes.
- Même thème visuel sur toutes les captures.
